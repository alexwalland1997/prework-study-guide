HTML

  • The head element contains information about the website
  • The body element represents the visible content shown to the user.
  • The heading label determines the size of the text and have to be done in numerical order.
  • The p label is used for paragraphs of text.
  • li is used for lists which can be unordered or ordered.
  • br is used to leave a space between text or images.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width determines how wide a item can be by pixels or percentage of screen.
  • Period is used before a class name to designate as class collector so anything with that class with gain the properties defined within it.
  • Colours can be done in word or hexidecimal format.

Git

Javascript

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A :adds all changes to current commit.
  • git commit -m "insert message": will update current branch with any changes not currently in it.
  • git pull main origin: pulls all modifications from the main branch from github repo.
  • git push origin branch-name: pushes local changes to that remote github branch.
